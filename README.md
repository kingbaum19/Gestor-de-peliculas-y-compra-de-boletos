# Gestor de pelicula y compra de boletos 
## ¿Qué herramientas utiliza este proyecto?
- Proyecto programado en C#
- Base de datos MySQL

## ¿De qué trata este proyecto?
Este proyecto es un gestor de peliculas de cine en donde se puede:
- Insertar información de películas dentro de una base de datos como una operación CRUD.
- Recrear el proceso de comprar un boleto de cine (Selección de película, selección de cantidad de asientos, reserva de asientos y compra).

## Requerimientos
Para que el proyecto funcione correctamente, debe importar la base de datos "cine.sql" que se encuentra en la carpeta "Base de datos" de este repositorio.
